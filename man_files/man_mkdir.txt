NAME
	garnet -- make a directory

SYNOPSIS
	garnet PARENT_DIRECTORY DIRECTORY_NAME
	garnet [--help]

DESCRIPTION
	This manual page documents version 1.0.0 of the garnet command.

	Create a directory in the PARENT_DIRECTORY with the name DIRECTORY_NAME, if they donot already exist

	optional parameters
		perm -- permissions for the created directory(default = 0777)
